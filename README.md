# Week 2 - CSS Personal Portfolio Website

## Project Overview

This project is a responsive personal portfolio website developed using **HTML5** and **CSS3** as part of the **Week 2 Internship Assignment**. The website presents my personal profile, education, technical skills, hobbies, and contact information in a clean, modern, and user-friendly design.

---

## Features

* Responsive design for desktop, tablet, and mobile devices
* External CSS for better code organization
* Professional and modern portfolio layout
* Circular profile picture
* Navigation menu with smooth section links
* Hover effects on navigation links and buttons
* Styled contact form
* Clean typography and color scheme
* Mobile-friendly interface

---

## Technologies Used
